##Overview
The purpose of this tutorial is to get you working with npm's socket.io and web sockets to implement a real-time collaborative application. This introduces the use-case where the server can send data to the client that is not in direct response to a request from that client.

## Prerequisites

Before you begin, ensure you have the following prerequisites installed:

- [Node.js](https://nodejs.org/): This project requires Node.js to run. You can download and install it from the official Node.js website.
- Socket.io: This project requires Socket.io module

##Installation
- [Node.js]
This project relies on Node.js for its runtime environment. Visit the [Node.js official website](https://nodejs.org/) for installation steps.

- [Socket.io]
run "npm install socket.io", this make the install the latest version of the socket.io version for the project




##Testing 
to test the assignment follow the steps:
- run the command: node server.js
- ensure "server.js" is in the current working directory 
- open the link displayed in the terminal twice: "http://localhost:3000/index.html"
- using the arrow key move the box in the first tab
- switch to the second window observe the box ha been moved exactly like how the box was moved in the first tab
