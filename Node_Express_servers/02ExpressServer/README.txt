##Overview
This tutorial relates to the course notes and lectures on "Introducing Express". You can learn more about express.js at this link: https://expressjs.com The notes may also contain some older chapters from books used in the past for the course, that explain some basics of express.js.

## Prerequisites

Before you begin, ensure you have the following prerequisites installed:

- [Node.js](https://nodejs.org/): This project requires Node.js to run. You can download and install it from the official Node.js website.
- Socket.io: This project requires Socket.io module

##Installation
- [Node.js]
This project relies on Node.js for its runtime environment. Visit the [Node.js official website](https://nodejs.org/) for installation steps.

- [Express.js]
run "npm install express", this make the install the latest version of the express version for the project

- [morgan]
run "npm install morgan", this make the install the latest version of the morgan version for the project




##Testing 
to test the assignment follow the steps:
- run the command: node 02_express_static_server.js
- ensure "02_express_static_server.js" is in the current working directory 
- open the link displayed in the terminal twice: "http://localhost:3000/index.html"
- go to the search bar and type "http://localhost:3000/table.html"
- the webpage show change to that html page
- go to the search bar and type "http://localhost:3000/morning.html"
- the webpage show change and display the error message "404: OOPS YOU BROKE THE INTERNET"
- return to the terminal and observe the log that is being tracked by the "morgan module"
